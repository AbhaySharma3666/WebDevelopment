AJAX = Asynchronous JavaScript and XML;

In short; AJAX is about loading data in the background and display it on the webpage, without reloading the whole page.

jQuery provides several methods for AJAX functionality.

With the jQuery AJAX methods, you can request text, HTML, XML, or JSON from a remote server using both HTTP Get and HTTP Post - And you can load the external data directly into the selected HTML element of your web pages.
