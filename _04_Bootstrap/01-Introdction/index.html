Welcome to Day 1

This is a simple HTML page styled with Bootstrap.

Bootstrap container hierarchy is {container ==> row ==> column.}

In Bootstrap, built-in classes help design websites faster and easier.

Bootstrap makes your website responsive, meaning it works well on all screen sizes.

You can create modern designs without writing custom CSS from scratch.

It includes a powerful grid system and many UI components like buttons, cards, navbars, etc.

Bootstrap also supports utility classes like margin, padding, color, and typography.

To use Bootstrap components like modals, carousels, and dropdowns, you can include Bootstrap JS.

You can customize Bootstrap with your own styles if needed.
